1. Preconditions On Clean VM

- repo is present at a known source path
- one real values file is prepared from `ops/templates/install_values.example.json`
- PostgreSQL role/database connectivity is ready for the `DB_URL` in that values file
- TLS/nginx prerequisites already exist for:
  - `SERVER_NAME`
  - `SSL_CERT_PATH`
  - `SSL_KEY_PATH`
- target `APP_DIR` does not already exist
- clean VM check must pass:
  - `/etc/gridflow` absent
  - `/etc/systemd/system/gridflow-app.service` absent
  - `/etc/systemd/system/upbit-bot.service` absent
  - `/etc/systemd/system/orderlens-ops.service` absent
  - `/etc/nginx/sites-available/gridflow` absent
  - target `APP_DIR` absent
