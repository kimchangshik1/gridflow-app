# GridFlow NDA Template

기준
- 작성 기준: 2026-04-21 UTC
- 성격: 법률 자문이 아닌 실무용 초안
- 용도: 판매 전 코드, 문서, 데모 자료 열람 절차에 맞춘 간단한 비밀유지 합의 초안

## 1. 목적

본 합의서는 GridFlow 관련 제품 설명, 데모, 문서 검토, 코드 또는 전달 범위 협의를 위해 제공되는 비공개 정보를 보호하는 것을 목적으로 한다.

## 2. 비밀정보 정의

비밀정보에는 아래 항목이 포함된다.

- 제품 문서, 데모 자료, 설치/운영 문서
- 코드, 설정 구조, 시스템 구성 정보
- 가격안, handoff 범위, 지원 조건
- 실사 과정에서 제공되는 증거 자료, 캡처, 운영 설명 자료
- 구두, 문서, 화면 공유, 파일 전달 등 어떤 방식으로 제공되든 비공개로 표시되거나 비공개로 보는 것이 합리적인 정보

다만 실제 secret 원문, 고객별 운영값, 내부 접근 자격증명은 별도 합의 없이는 열람 범위에 포함하지 않는다.

## 3. 사용 목적 제한

수령자는 비밀정보를 아래 목적에 한해 사용할 수 있다.

- GridFlow 도입 검토
- 기술 실사
- 가격 및 handoff 범위 협의
- 계약 체결 여부 판단

수령자는 위 목적 외의 사용, 복제, 역이용, 제3자 영업 활용을 해서는 안 된다.

## 4. 제3자 제공 금지

수령자는 제공자의 사전 서면 동의 없이 비밀정보를 제3자에게 제공하거나 공개할 수 없다. 다만 내부 검토가 필요한 경우, 동일한 수준의 비밀유지 의무를 부담하는 임직원 또는 자문인에게 필요한 범위 안에서만 공유할 수 있다.

## 5. 예외

아래 정보는 비밀정보에서 제외한다.

- 수령 시점에 이미 적법하게 알고 있던 정보
- 공개 시점에 이미 공개되어 있던 정보
- 수령자 책임 없는 사유로 공개된 정보
- 제3자로부터 적법하게 취득한 정보
- 법령 또는 법원, 감독기관 요구로 공개가 필요한 정보

법령상 공개가 필요한 경우 가능한 범위에서 사전에 상대방에게 통지한다.

## 6. 보관 / 폐기

수령자는 비밀정보를 합리적인 보안 수준으로 보관해야 한다. 검토가 끝나거나 상대방이 요청하는 경우, 제공된 문서와 파일을 반환하거나 폐기하고, 요청이 있으면 폐기 완료 사실을 서면 또는 이메일로 확인한다.

## 7. 기간

본 합의서의 효력은 서명일로부터 시작한다.

- 계약 검토 기간: `[NDA_REVIEW_TERM]`
- 비밀유지 의무 기간: `[NDA_CONFIDENTIALITY_TERM]`

정확한 기간은 거래 조건에 맞춰 당사자가 협의해 확정한다.

## 8. 관할 / 분쟁

준거법과 관할 법원은 아래 placeholder에 따라 별도 확정한다.

- 준거법: `[GOVERNING_LAW]`
- 관할 법원 또는 분쟁 해결 방식: `[JURISDICTION_OR_DISPUTE_METHOD]`

## 9. 열람 절차 메모

실무상 판매 전 열람은 아래 순서로 진행할 수 있다.

1. NDA 체결
2. one-pager, FAQ, handoff scope 등 buyer-facing 문서 열람
3. 필요 시 demo script 기준 데모 진행
4. 필요 시 추가 evidence 또는 clean delivery set 설명 제공

실제 코드 원문, 내부 evidence 원본, secret 원문은 별도 합의된 범위에서만 제공한다.

## 10. 서명란

- 제공자: `[DISCLOSING_PARTY]`
- 수령자: `[RECEIVING_PARTY]`
- 담당자명: `[NAME]`
- 직함: `[TITLE]`
- 서명: `[SIGNATURE]`
- 날짜: `[DATE]`
