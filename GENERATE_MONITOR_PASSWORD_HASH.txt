GENERATE_MONITOR_PASSWORD_HASH

Purpose
- Generate a bcrypt hash string for `monitor_auth.json`.
- The output goes into the `password_hash` field or `{{MONITOR_PASSWORD_HASH}}` replacement value.

Usage
```bash
python3 ops/generate_monitor_password_hash.py --password 'your-password'
printf 'your-password\n' | python3 ops/generate_monitor_password_hash.py
python3 ops/generate_monitor_password_hash.py
```

Input
- `--password`: plain-text password
- or stdin
- or interactive prompt if stdin is a TTY

Output Use
- Put the stdout hash string into:
  - `ops/templates/install_values.example.json` replacement workflow as `MONITOR_PASSWORD_HASH`
  - `monitor_auth.json` field `password_hash`

Notes
- Empty passwords fail.
- Output is a bcrypt hash string compatible with `app/monitor/auth.py`.
