4. Placeholder Meaning

- {{DRY_RUN}}
  Meaning: global dry-run switch read by app/core/config.py
  Location: gridflow.env
  Required/Optional: optional

- {{DB_URL}}
  Meaning: primary PostgreSQL connection string for web, bot, monitor, and backup script
  Location: gridflow.env
  Required/Optional: required

- {{JWT_SECRET}}
  Meaning: value present in the current installed env file, but runtime reference is unproven in current inspected code
  Location: gridflow.env
  Required/Optional: unproven / currently treated as installed-file placeholder only

- {{GRIDFLOW_ALERT_WEBHOOK_URL}}
  Meaning: outbound webhook endpoint for health-alert push notifications
  Location: gridflow.env
  Required/Optional: optional

- {{MONITOR_EXCHANGE}}
  Meaning: exchange selector consumed by monitor config
  Location: monitor_config.json
  Required/Optional: required

- {{MONITOR_API_KEY}}
  Meaning: monitor exchange API key
  Location: monitor_config.json
  Required/Optional: required

- {{MONITOR_API_SECRET}}
  Meaning: monitor exchange API secret
  Location: monitor_config.json
  Required/Optional: required

- {{MONITOR_TIER}}
  Meaning: monitor tier value read by app/monitor/tier.py
  Location: monitor_config.json
  Required/Optional: optional

- {{MONITOR_PASSWORD_HASH}}
  Meaning: stored bcrypt password hash for monitor login
  Location: monitor_auth.json
  Required/Optional: required

5. Install-Time Replacement Values

- {{DRY_RUN}}
  Replace with: `true` or `false`
  Manual input needed: yes

- {{DB_URL}}
  Replace with: PostgreSQL URL for the target environment
  Expected shape from current install: `postgresql://USER:PASSWORD@HOST:PORT/DBNAME`
  Manual input needed: yes

- {{JWT_SECRET}}
  Replace with: deployment-specific secret string if this key is kept in the install payload
  Manual input needed: yes

- {{GRIDFLOW_ALERT_WEBHOOK_URL}}
  Replace with: deployment-specific webhook URL or leave empty if alerts are intentionally disabled
  Manual input needed: yes

- {{MONITOR_EXCHANGE}}
  Replace with: current allowed/live-observed exchange value such as `upbit` or `bithumb`
  Manual input needed: yes

- {{MONITOR_API_KEY}}
  Replace with: monitor exchange API key
  Manual input needed: yes

- {{MONITOR_API_SECRET}}
  Replace with: monitor exchange API secret
  Manual input needed: yes

- {{MONITOR_TIER}}
  Replace with: `lite`, `pro`, or `signature`
  Manual input needed: yes if tier is used; otherwise choose a default intentionally

- {{MONITOR_PASSWORD_HASH}}
  Replace with: bcrypt hash string, not plain password
  Manual input needed: yes
