- 해석 한계:
  - raw row는 최종 KRW balance 한 줄만 보여준다.
  - BUY 전/후, SELL 후 재증가까지의 시계열은 이 extract만으로 닫히지 않는다.

### 2. qty / position bridge

- sandbox_orders 스키마에는 `qty` 컬럼이 없다.
- 따라서 DB raw pack 기준으로는 `amount_krw` / `price` / `status`까지만 직접 확인된다.
- 포지션 수량 변화는 기존 runtime JSON / UI 캡처를 함께 봐야 한다.

## Missing

### 1. sandbox_balances 시계열

- BUY 직전 / BUY 직후 / SELL 직후 KRW 변화 row 묶음은 이번 extract에 없다.

### 2. sandbox position DB table bridge

- 현재 확인 범위에서는 guest 포지션 수량 변화를 직접 보여주는 별도 DB raw table은 포함하지 않았다.
